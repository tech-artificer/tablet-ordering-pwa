# CASE FILE: Tablet Ordering PWA (The Mouthpiece)
**Status:** User Interface / Client Side
**Primary Stack:** Vue.js (Nuxt 3), TypeScript, Tailwind CSS

## 🎯 Goal
A high-speed, intuitive interface for tableside ordering. It must be reliable even on tablet-grade hardware.

## 🧩 Subjects
- **State Management:** Using Pinia to track order items before submission.
- **PWA Features:** Offline caching and "Install to Home Screen" capabilities.
- **Backend Sync:** Interfacing with `woosoo-nexus` for menu fetching and order submission.
- **UI/UX:** Responsive layouts for various tablet sizes using Tailwind.

## 📜 Coding Standards
- Composition API (Script Setup) with strict TypeScript.
- Optimistic UI updates (show the item in the cart instantly).
- Modular Vue components for menu items, categories, and checkout.
