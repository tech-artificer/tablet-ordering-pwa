- Ensure you do NOT modify `nuxt.config.ts` or any service worker files in this task.
- Run verification commands:
  - `npm ci`
  - `npm run lint || true`
  - `npm run test || true`
- For offline testing, use browser tools or Playwright to simulate offline scenarios. Provide explicit steps in the PR to reproduce any offline tests.
