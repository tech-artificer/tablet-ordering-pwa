# Agent Rules — Tablet Ordering PWA

## Scope

This repo owns only the tablet client.

Do not modify Nexus, Laravel, POS, or Docker gateway files from this repo.

## Hard Rules

- Do not hardcode API/Reverb hosts.
- Do not change PWA caching/update behavior without validation.
- Do not force reload during active session/order submission.
- Do not change persisted store keys without migration strategy.
- Do not merge UI redesign with deployment/runtime fixes.

## Required Before Merge

```bash
npm run typecheck
npm run lint
npm run test
npm run build
npm run generate
```

## Critical Runtime Contract

Integrated production runtime config comes from:

```txt
/runtime-config.js
window.__APP_CONFIG__
```

The app must load `/runtime-config.js` before relying on API/Reverb config.
